## 📝 このリポジトリについて

Google STEP 教育コースでは、主に以下のスキルを中心に学びました：

- データ構造とアルゴリズム
- 効率的なコーディングとデバッグ方法
- 技術的プレゼン、キャリア相談、面接対策
- コンピュータの基礎的な仕組み理解

各フォルダ（例：`Week1/`）には、その週の課題コード、メモ、改善案などを整理しています。
